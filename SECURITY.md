# Security Policy

## Supported versions

FIO is pre-1.0 (`0.x`). Only the latest published release of each package
(**FIO**, **FIO.Http**, **FIO.Sockets**, **FIO.WebSockets**) receives security
fixes. Older `0.x` releases are not patched — upgrade to the latest version.

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report privately via GitHub's
[private vulnerability reporting](https://github.com/fs-fio/fio/security/advisories/new),
or email **hey@itsdaniel.dk**.

Include, where possible:

- the affected package and version,
- a description of the issue and its impact,
- steps to reproduce or a proof of concept.

You can expect an acknowledgement within a few days. Once a fix is ready, a
patched release is published to NuGet and the advisory is disclosed.
